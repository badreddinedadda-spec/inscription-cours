#  InscriptionCours 

Application web Spring Boot de gestion des inscriptions & courses.

## Technologies
- Java 21 + Spring Boot 3.4
- Spring Security + BCrypt
- Spring Data JPA + MySQL
- Thymeleaf + CSS vanilla
- Mailtrap SMTP
- Déployé sur Railway

## Fonctionnalités
- Formulaire d'inscription public
- Tableau de bord admin sécurisé
- Validation/refus des inscriptions
- Notifications email automatiques
- Éditeur de contenu CMS
- Réinitialisation de mot de passe

## Accès admin
Interface d'administration sécurisée - accès restreint.
